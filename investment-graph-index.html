🔬 Alpha Lab · Investment Graph POC

同一份 mock 数据 (16 家半导体/AI 公司 + 26 条关系) 在 7 种前端库里的不同呈现. 挑一个你最喜欢的.

已实现
🎈

force-graph (current)

弹簧漂浮圆球 + Canvas + d3-force 物理模拟. 拖一个球带动邻居, 有惯性有阻尼有碰撞体积. 视觉最"活".

擅长: 中小图 (< 500 节点) · 物理感 · 快速原型
 强推
🧠

Cytoscape.js 图算法

带大脑的图库. 一行代码就能拿到 "TSM 上游 3 跳产业链" (BFS). 内置 20+ 布局 (环形/树/放射) + 最短路径 + PageRank.

擅长: 需要"点一个球高亮整条产业链" 这类图分析
 强推
🎯

D3 Circle Packing 层级套娃

母球套子球. 半导体产业 → AI 芯片链 / HBM 存储 / CoWoS → 具体公司. 点大圆下钻放大, 视觉最抓人.

擅长: 表达"归属层级" · 上台演示 (PPT 直接用截图)
📐

Sigma.js WebGL

WebGL 硬件加速. 学术论文常见风格, 平面 2D, 节点带 label. 5000 节点也 60fps 流畅.

擅长: 大规模 (> 1000 节点) · 学术/工业感
🏛️

AntV G6 阿里出品

阿里数据可视化团队作品. 中文文档全, 视觉专为**金融/商业中台**设计. 边有贝塞尔曲线 + 箭头, 节点带 badge 多状态.

擅长: 中文项目 · 需要商业感 · 内置金融组件
💳

React Flow 卡片式

每个节点是一张**卡片** (可以是 React 组件). 节点里嵌 ticker + mock 股价 + 迷你 K 线 + 涨跌色. 从"圆球" 升级到 "股价 dashboard".

擅长: 想让节点自身携带丰富信息
🕸️

vis-network 经典可靠

经典网络图库, API 直白, 边可以是曲线, 支持层级布局 (树状). 配置直接一个大 JSON.

擅长: 快速搭 MVP · 配置化 · 稳定不折腾
